First of all what is HTML Template?

HTML Template is a collection of html files that can be used as standalone files to put them directly on your server or implement the code in any programming language like php or ASP.NET, etc...

Why SuperFine ?

As SuperFine is a responsive MultiPurpose HTML5 Template, you can use it in all of your projects, this is because it is fully responsive and compatible with any device, PC or tablets, and also with any OS, Also with our tons of features and endless possibilities you can create all your websites with different look and feel for each one individually.

Getting Statred

To install this HTML template you will need to upload all files and folders using an FTP client. We recommend using FileZilla FTP client. To upload your template you will need the following:

  • Domain
  • FTP Username
  • FTP Password

Once connected make sure to upload All files and folders within the "SuperFine" folder of your download package.

Thing You Should Know before you start
  • After downloading the template you'll find that all images have width and height written on them. For best appearance resize your images to be the same width and height.
  • Do not start building from scratch , even if you want that , we have included a blank.html page that contains the header and footer only and you can add your inner content in the empty content section.

	function init_map(){
		var myOptions = {zoom:14,center:new google.maps.LatLng(40.805478,-73.96522499999998),mapTypeId: google.maps.MapTypeId.ROADMAP};
		map = new google.maps.Map(document.getElementById("map_canvas"), myOptions);
		marker = new google.maps.Marker({map: map,position: new google.maps.LatLng(40.805478, -73.96522499999998)});
		infowindow = new google.maps.InfoWindow({content:"<div class='noScroll'><b>Headquarter</b><br/>2880 Broadway<br/> New York</div>" });

			infowindow.open(map,marker);
		});
		infowindow.open(map,marker);
	}
	google.maps.event.addDomListener(window, 'load', init_map);
</script>

    Change the center: new google.maps.LatLng & map: map,position: new google.maps.LatLng values to your new values.

style.css : (the main css file), This file contains all general styling, such as positioning, floatings, anchor tag colors, font-sizes, etc. Keep in mind, that these values might be overridden somewhere else in the file.

light.css and dark.css : for the light and dark skins

rtl.css file contains all of the specific styling for the RTL support

Light & Dark Theme

  • There are two ways to select the theme color (Light and Dark):

    • From HTML file Directly: open the HTML file and in the page head tag find: 
      	<link id="theme_css" rel="stylesheet" href="assets/css/light.css">

      Change light.css to dark.css.

      You will need to do this step with all other HTML files
    • From the script.js file: Check the image below:

      Change $theme value from light to dark.

      No need to change all HTML files, this option will be applied on all pages.

Choose Skin Color

  • There are two ways to select the skin color (from the available 12 skin css files in assets/css/skins foler):

    • From HTML file Directly: open the HTML file and in the page head tag find: 
      	<link id="skin_css" rel="stylesheet" href="assets/css/skins/default.css">

      Change default.css to any other skin name from 1 to 12.css.

      You will need to do this step with all other HTML files
    • From the script.js file: Check the image below:

      Change $skin value from default to 1 or 2 ... to 12.

      No need to change all HTML files, this option will be applied on all pages.

Wide & Boxed Layout

  • There are two ways to select the layout style (Wide and Boxed):

    • From HTML file Directly: open the HTML file and find: 
      	<div class="pageWrapper">

      add a new class: boxed to be <div class="pageWrapper boxed">.

      also add a new class: bg1 or bg2, ... to bg13 for the body tag, to be <body class="bg1">.

      You will need to do this step with all other HTML files
    • From the script.js file: Check the image below:

      Change $layout value from wide to boxed.

      Change $bodyclass value from bg1 to bg2, ... to bg13. This will only be applied for boxed layout

      No need to change all HTML files, this option will be applied on all pages.

Choose Your Prefered Shape.

  • There are two ways to choose your prefered shape (new-angle, round, border5px, square, right-angle, left-angle, top-angle, bottom-angle):

    • From HTML file Directly: open the HTML file and find any element with class "shape" Ex: 
      	<div class="shape">

      Remove the class "shape" and instead of it add a class from the following: new-angle or round or border5px or square or right-angle or left-angle or top-angle or bottom-angle.

      You will need to do this step with all other HTML files
    • From the script.js file: Check the image below:

      Change $shape value from new-angle to one of the values decribed beside it.

      No need to change all HTML files, this option will be applied on all pages.

RTL Support

  • From HTML file Directly: open the HTML file and before the  close of <head> tag add: 
    	<link rel="stylesheet" href="assets/css/rtl.css">
    You will need to do this step with all other HTML files
  • From the script.js file: Check the image below:

    Change $rtl value from 0 (false) to 1(true).

    No need to change all HTML files, this option will be applied on all pages.
